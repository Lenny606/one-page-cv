# One Page CV - Astro SSG

A premium, localized (EN) one-page CV built with Astro, Tailwind CSS, and GSAP. Designed for high performance and smooth interactive experiences.

## 🚀 Features

- **Astro SSG**: Blazing fast static site generation.
- **Tailwind CSS**: Modern utility-first styling.
- **GSAP**: Professional-grade animations.
- **GitHub Actions**: Automated deployment to GitHub Pages.
- **Project Management SKILL**: Integrated task monitoring and data management.

## 🛠️ Project Structure

- `src/`: Application source code (Astro components, styles).
- `temp/`: Task logs and project references.
- `.github/workflows/`: CI/CD deployment pipeline.
- `.agent/skills/`: Custom agent-based management tools.

## 📦 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start development server**:
    ```bash
    npm run dev
    ```
3.  **Build for production**:
    ```bash
    npm run build
    ```

## 📝 Task Monitoring

The project uses a custom monitoring system. All major changes are logged in `temp/task-log.md` as per the instructions in `.agent/skills/project-management/SKILL.md`.
